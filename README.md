# CyberBots - Directorio de Bots de Discord

El directorio definitivo de bots para Discord con diseno neon/cyberpunk.

## Caracteristicas

- Lista todos los bots con tarjetas neon animadas
- Bot destacado con seccion especial
- Boton Invitar directo a Discord OAuth2
- Sistema de votos por bot
- Buscador de bots por nombre o etiqueta
- Formulario para anadir tu propio bot
- Diseno neon rainbow con particulas animadas
- SEO optimizado para Google

## Stack Tecnico

- Frontend: React + Vite + TypeScript + Tailwind CSS + Framer Motion
- Backend: Express 5 + Node.js
- Base de datos: PostgreSQL + Drizzle ORM
- API: OpenAPI 3.1 + Orval (codegen)
- Monorepo: pnpm workspaces

## Instalacion

  pnpm install
  pnpm --filter @workspace/api-server run dev
  pnpm --filter @workspace/discord-bots run dev
